问题记录

  1. No valid crumb was included in the request
    解决方案：
    • 在jenkins 的Configure Global Security下 , 取消“防止跨站点请求伪造（Prevent Cross Site Request Forgery exploits）”的勾选

  2. 插件下载过慢，失败
    解决方案：

    • 修改下载源,jenkins_home/updates/default.json

  3. mvn 未找到命令
    解决方案：
